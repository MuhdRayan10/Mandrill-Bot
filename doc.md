### Moderation Commands

`/kick` - To kick a user from the server

`/ban` - To ban a user from the server

`/tempban` - To ban a user from the server for x minutes and then unban

`/softban` - To ban and quickly unban (to remove messages) a user from the server

`/mute` - To mute a user in the server

`/unmute` - To unmute a user in the server

`/tempmute` - To temporarily mute a user for x minutes

`/purge` - To delete x messages from the current channel
